Quando você consome a informaçao da internet do servidor. normalmente sao arquivo estáticos, html, css, javascript , imagens… e preocupaçao e fazer um site com design responsivo ou responsividade.

O que são Servers?

Um servidor é um sistema de computador ou software que fornece serviços, recursos ou dados a outros computadores, conhecidos como clientes, em uma rede.

Como é uma estrutura básica de um HTML?

A estrutura básica do HTML é composta pelas tags , <html> , <head> e <body> . O <!DOCTYPE html> declara o tipo de documento, o <html> envolve todo o conteúdo, o <head> contém metadados sobre a página e o <body> contém o conteúdo visível.

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas (ou numeradas) usam números, letras ou algarismos romanos para indicar a sequência dos itens, enquanto listas não ordenadas (ou com marcadores) empregam marcadores visuais, como círculos ou quadrados, para apresentar os itens sem uma ordem específica.

Web Scraping

Web scraping é o processo de extrair dados de sites da web usando um programa ou script.

O que são <h1> a <h6>?

São tags de cabeçalho no HTML — usadas para criar títulos e subtítulos na sua página. Elas indicam níveis de importância de conteúdo.

O que é a tag <p>?

Em HTML, a tag <p> é usada para definir um parágrafo de texto. Ela representa um bloco de texto que é separado de outros blocos por espaços em branco verticais e, geralmente, recuo na primeira linha.

O que é a tag <mark>?

Em HTML, a tag <mark> é utilizada para destacar ou realçar texto, geralmente dentro de um contexto específico, como um parágrafo ou uma lista.

O que é a tag <smal>?

O elemento html <smal> é utilizado para exibir texto com uma fonte menor do que o tamanho padrão,

O que é a tag <i>?

Em HTML, a tag <i> é usada para definir uma parte do texto com uma voz ou humor diferente, geralmente exibida em itálico.

O que é a tag <u>?

Em HTML, a tag <u> é usada para sublinhar texto, mas no HTML5, ela recebeu um significado semântico mais amplo, representando uma anotação não textual sobre o texto, em vez de simplesmente um efeito visual de sublinhado.

O que é a tag <strong>?

A tag <strong> em HTML é usada para definir um texto com importância forte ou ênfase,

O que é a tag <ol>

O elemento Html <ol> define uma lista ordenada, onde os itens são apresentados em sequência numérica ou alfabética.

O que é a tag <ul>?

Em elemento <ul> representa uma lista não ordenada. É usado para agrupar uma série de itens, onde a ordem em que eles aparecem não é significativa.

O que é a tag <li>?

Em HTML, a tag <li> O elemento HTML <li> (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista. Este item deve estar contido em um elemento pai: uma lista ordenada (<ol>), uma lista desordenada (<ul>)

O que é a tag <a>?

O elemento <a> em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada precisará indicar o destino do link.

O que é a tag <hr>?

O elemento HTML <hr> representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção).

O que é a tag <sub>?

O elemento HTML <sub> é usado para definir texto subscrito, que aparece um pouco abaixo da linha de texto normal e geralmente com um tamanho de fonte menor. É comumente utilizado em fórmulas químicas (como H₂O) ou em expressões matemáticas para indicar índices ou expoentes.

O que é a tag <sup>?

O elemento HTML especifica o texto embutido que deve ser exibido como sobrescrito por razões exclusivamente tipográficas. Sobrescritos geralmente são renderizados com uma linha de base elevada e texto menor.<sup>

O que é a tag <blockquote>?

O elemento <blockquote> em HTML é utilizado para marcar citações longas de outro conteúdo. Ele indica que o texto dentro do elemento é uma citação e, geralmente, é exibido com recuo nas margens, tanto na visualização quanto na impressão. Além disso, o atributo cite pode ser usado para fornecer um link para a fonte da citação.

O que é a tag <font>?

A tag <font> em HTML serve para especificar o tipo, tamanho e cor da fonte de um texto em uma página web. No entanto, essa tag foi descontinuada no HTML5 e não é mais recomendada para uso. Em vez dela, é preferível utilizar CSS

O que é a tag <del>?

O elemento HTML <del> é usado para marcar texto que foi removido de um documento. Ele geralmente é exibido com um risco por cima do texto. É útil para mostrar alterações em documentos ao longo do tempo ou para indicar que um trecho de texto não é mais relevante.

O que é a tag <abbr>?

A tag HTML <abbr> é usada para definir uma abreviação ou acrônimo em um documento HTML. Ela serve para indicar que um trecho de texto é uma abreviação, e opcionalmente, fornecer sua expansão completa através do atributo title.
